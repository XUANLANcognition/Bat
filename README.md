# Bat

## 准备环境

1. 克隆项目
```
git clone https://github.com/XUANLANcognition/bat.git
cd bat
```
2. 安装环境
```
apt install pipenv
pipenv install
pipenv shell
pip install -e .
```

## Modules
1. [SYN scan](https://github.com/XUANLANcognition/bat/blob/master/bat/syn_scan/README.md)
```
python -m bat syn [IP] [PORT] 
```
